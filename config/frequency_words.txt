# 关键词语法：
# 关键词 普通关键词，标题包含即匹配
# /正则/ 正则表达式匹配（自动忽略大小写）

# +关键词 必须词，所有必须词都要匹配才算匹配
# !关键词 过滤词，匹配则排除该条新闻（仅限当前词组）
# @数字 限制该词组最多显示多少条

# 显示名称优先级：
# 1. 有组别名 → 显示组别名
# 2. 没有组别名 → 显示各行别名拼接（用 " / " 连接）
